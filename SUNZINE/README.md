**SUNZINE** is a modern, minimalist digital agency portfolio and creative showcase highlighting design work, photography, and brand identity projects.

---

## ✨ Features

- **Filterable Work Showcase**: Category navigation bar (All, Design, Branding, Photography, Art).
- **Responsive Masonry-Style Grid**: Flexible image card gallery for project portfolios.
- **Social Media Connectivity**: Integrated header social icon group.
- **Clean Typography & Active Indicators**: Teal bottom indicator bar for active navigation tabs.

---

## 🛠️ Tech Stack

- **Framework**: Bootstrap 5.3.3.
- **Icons**: FontAwesome 6.4.
- **Styling**: Minimalist custom CSS with teal active accents (`#00c9a7`).

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SUNZINE.git
   cd SUNZINE
   ```

2. **Open `index.html`** in any web browser.
