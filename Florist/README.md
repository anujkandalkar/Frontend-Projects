**Florist** is an elegant, modern e-commerce landing application tailored for fresh flower boutiques, bouquet customizers, and gift shops.

---

## ✨ Features

- **Hero Carousel**: Full-bleed Bootstrap carousel with custom font typography (*Pacifico* & *Poppins*).
- **Service Highlights**: Icon feature row highlighting free delivery, fresh blooms, and 24/7 client support.
- **Product Showcase**: Filterable grid layout for seasonal bouquets, vase arrangements, and custom gifts.
- **Shopping Cart Counter**: Integrated navbar cart icon badge and currency total display.
- **Newsletter Subscription**: Clean inline footer subscription input field.

---

## 🛠️ Tech Stack

- **Framework**: Bootstrap 5.3.3.
- **Icons & Fonts**: FontAwesome 6.5, Google Fonts (*Pacifico*, *Poppins*).
- **Styling**: Custom CSS overrides for boutique branding and subtle animations.

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Florist.git
   cd Florist
   ```

2. **Open `index.html`** directly in any web browser.
