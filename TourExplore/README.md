# 🏔️ TourExplore - Maharashtra Travel Showcase

**TourExplore** is a dedicated travel destination portal showcasing top tourist destinations, cultural spots, and hill stations across Maharashtra, India.

---

## ✨ Features

- **Hero Banner**: Full-width header introducing Maharashtra's heritage and landscapes.
- **Scrollable Tour Packages**: Horizontal card list featuring Chhatrapati Sambhajinagar, Nashik, Saputara, Mahabaleshwar, Ganpatipule, Kolhapur, and Malvan.
- **Summary Matrix Table**: Comparison table listing locations alongside key historical and natural attractions.
- **Responsive Layout**: Fluid CSS design adapted for various screen sizes.

---

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5 & Custom CSS3.
- **Media**: Local image assets showcasing regional destinations.

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/TourExplore.git
   cd TourExplore
   ```

2. **Open `index.html`** in any web browser.
