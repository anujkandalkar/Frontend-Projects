# 🥗 VEGEFOODS - Organic Grocery & Produce Store

**VEGEFOODS** is a vibrant, modern e-commerce storefront designed for organic farms, fresh fruit markets, and local vegetable delivery services.

---

## ✨ Features

- **Announcement Topbar**: Contact phone, email, and free shipping notification header.
- **Organic Hero Section**: Vibrant hero banner with CTA button for product discovery.
- **Service Benefits Grid**: Highlights free shipping, fresh packaging, superior quality, and customer support.
- **Featured Produce Catalog**: Product cards featuring price tags, discount badges, and quick-add actions.
- **Customer Testimonials & Stats**: Counter metrics and customer reviews block.

---

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5 & Custom CSS3.
- **Design Aesthetic**: Organic green color palette (`#82ae46`) with card elevation.

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/VEGEFOODS.git
   cd VEGEFOODS
   ```

2. **Open `index.html`** in any web browser.
