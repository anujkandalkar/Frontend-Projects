## ✨ Key Features

- 📱 **100% Mobile & Tablet Responsive:** Crafted using modern CSS Flexbox and CSS Grid to ensure flawless display across all screen sizes.
- 🎨 **Modern UI/UX Design:** Micro-animations, smooth hover states, curated color palettes, and glassmorphism elements.
- ⚡ **Lightweight & Fast Load Times:** Pure native web technologies with zero bloat or unnecessary dependencies.
- 🔍 **SEO & Accessibility Optimized:** Semantic HTML5 tags (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`), structured meta tags, and high-contrast color ratios.
- 🛠️ **Developer Friendly:** Clean, well-commented, and easily customizable codebases.

## 🚀 Getting Started

### Prerequisites

All you need is a web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge) and a code editor like [Visual Studio Code](https://code.visualstudio.com/).

### Installation & Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/anujkandalkar/Frontend-Projects.git
   ```

2. **Navigate to the repository folder:**
   ```bash
   cd Frontend-Projects
   ```

3. **Run any project:**
   - Simply open the `index.html` file of any project folder directly in your browser.
   - *Optionally*, use the **Live Server** extension in VS Code for live reloading:
     - Right-click `index.html` inside any project folder (e.g., `Academics/index.html`) -> **Open with Live Server**.

## 🌐 Deploying to GitHub Pages (Optional)

You can easily host these projects live using **GitHub Pages**:

1. Go to your repository on GitHub: `https://github.com/anujkandalkar/Frontend-Projects`
2. Navigate to **Settings** > **Pages**.
3. Under **Build and deployment** > **Source**, choose **Deploy from a branch**.
4. Select branch **`main`** and folder **`/ (root)`**, then click **Save**.
5. Your projects will be available live at `https://anujkandalkar.github.io/Frontend-Projects/`!
