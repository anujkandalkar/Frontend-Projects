**Academics** is a feature-rich, responsive frontend portal for educational institutions, online academies, and e-learning platforms.

---

## ✨ Features

- **Header Topbar**: Contact information, social links, and quick login/register modal navigation.
- **Authentication Pages**: Dedicated responsive `login.html` and `register.html` portals with form validation styling.
- **Interactive Course Grid**: Course catalog cards highlighting categories, ratings, pricing tags, and instructor info.
- **Responsive Mobile Navigation**: Toggle-ready mobile navigation menu for seamless tablet and phone browsing.

---

## 🛠️ Tech Stack

- **Frontend**: Semantic HTML5, Vanilla CSS3 (Flexbox & Grid Layouts), FontAwesome 6 Icons.
- **Typography & Assets**: Custom Web Fonts & High Quality Educational Mockups.

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Academics.git
   cd Academics
   ```

2. **Open `index.html`** in any web browser or use a live server extension.
