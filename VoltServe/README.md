# ⚡ VoltServe - Professional Electrical Services Landing Page

**VoltServe** is a responsive electrical engineering and maintenance company portal, highlighting residential, commercial, and industrial power solutions.

---

## ✨ Features

- **Sticky Brand Header**: Orange brand header with responsive link navigation.
- **Service Categories Grid**: Equipment maintenance, residential wiring, bulb installations, and industrial setup cards.
- **About Us Section**: Corporate background with high-resolution imagery.
- **Customer Testimonials**: Client feedback cards with star ratings.
- **Zero Horizontal Scroll**: Pure CSS overflow protection preventing whitespace gaps on mobile screens.

---

## 🛠️ Tech Stack

- **Frontend**: HTML5, Embedded CSS3 (`overflow-x: hidden`, Flexbox, & Media Queries).
- **Icons & Assets**: Custom service icon graphics and transparent PNG assets.

---

## 🚀 Quick Start

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/VoltServe.git
   cd VoltServe
   ```

2. **Open `index.html`** in any web browser.
